# 快速开始指南

欢迎使用文档与项目管理平台！本指南将帮助您快速上手。

## 5 分钟快速启动

### 步骤 1：安装依赖

```bash
# 安装后端依赖
npm install

# 安装前端依赖
cd client
npm install
cd ..
```

### 步骤 2：启动应用

**方式一：使用启动脚本（推荐）**

```bash
./start.sh
```

**方式二：手动启动**

```bash
# 终端 1 - 启动后端
npm start

# 终端 2 - 启动前端
cd client
npm run dev
```

### 步骤 3：访问应用

打开浏览器访问：**http://localhost:5173**

### 步骤 4：注册账号

1. 点击"立即注册"
2. 填写用户名、邮箱和密码
3. 点击"注册"按钮

### 步骤 5：创建第一个项目

1. 登录后自动跳转到项目列表
2. 点击"+ 新建项目"按钮
3. 填写项目名称和描述
4. 点击"创建"

🎉 恭喜！您已经成功创建了第一个项目！

## 主要功能使用

### 📁 项目管理

**创建项目**
- 在项目列表页点击"+ 新建项目"
- 填写项目信息并提交

**查看项目**
- 点击项目卡片进入项目详情页
- 可以查看文档、任务和成员

### 📝 文档管理

**创建文档**
1. 进入项目详情页
2. 切换到"文档"标签
3. 点击"+ 新建文档"
4. 使用 Markdown 格式编写文档
5. 保存为草稿或直接发布

**编辑文档**
- 在文档详情页点击"编辑文档"
- 修改内容后保存

**发布文档**
- 在文档详情页点击"发布文档"
- 已发布的文档会出现在"已发布文档"页面

### ✅ 任务管理

**创建任务**
1. 进入项目详情页
2. 切换到"任务"标签
3. 点击"+ 新建任务"
4. 填写任务信息（标题、描述、优先级、截止日期）
5. 点击"创建"

**管理我的任务**
- 点击顶部导航的"我的任务"
- 查看分配给自己的所有任务
- 直接在列表中更新任务状态

### 👥 团队协作

**查看项目成员**
- 进入项目详情页
- 切换到"成员"标签
- 查看所有项目成员及其角色

## 常见问题

### Q: 忘记密码怎么办？
A: 目前版本暂不支持密码重置，请联系管理员或使用数据库工具修改。

### Q: 如何删除项目？
A: 在项目详情页，项目所有者可以删除项目（功能需要在前端添加删除按钮）。

### Q: 文档支持上传图片吗？
A: 当前版本支持 Markdown 引用外部图片链接。

### Q: 如何分配任务给其他成员？
A: 创建任务时可以选择负责人（需要先添加项目成员）。

### Q: 数据存储在哪里？
A: 所有数据存储在项目根目录的 `database.sqlite` 文件中。

## 技术支持

### 查看日志

```bash
# 后端日志
tail -f server.log

# 前端日志（浏览器控制台）
F12 打开开发者工具 → Console
```

### 重置数据库

```bash
# 停止服务
# 删除数据库文件
rm database.sqlite

# 重新启动服务（会自动创建新数据库）
npm start
```

### 端口冲突

如果端口被占用，可以修改配置：

**后端端口**（修改 `.env`）：
```
PORT=3001
```

**前端端口**（修改 `client/vite.config.ts`）：
```typescript
export default defineConfig({
  server: {
    port: 5174
  }
})
```
